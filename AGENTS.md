# Codex Adapter

Este repositorio usa [`CONTEXT.md`](./CONTEXT.md) como fuente unica de verdad para contexto durable.

Instrucciones:
- leelo antes de trabajar;
- prefiere actualizar ese archivo en lugar de crear memoria duplicada;
- separa `HECHO_VERIFICADO`, `INFERENCIA`, `HIPOTESIS_DE_TRABAJO` y `PENDIENTE`;
- corrige con respeto cuando el usuario o la memoria se contradigan con la evidencia;
- nunca inventes informacion para completar una narrativa.
